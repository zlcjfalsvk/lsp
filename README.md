# [LSP(Language Server Protocol)](https://microsoft.github.io/language-server-protocol/)

- Setup 
  - `npm install`
- Start
  - `npm run start`


### What is LSP?
편집기 또는 IDE 에서 자동 완성, 정의 이동 등을 사용하기 위한 구현을 목적으로 한 프로토콜 <br />

### Required Knowledge

- IPC
- JSON-RPC
